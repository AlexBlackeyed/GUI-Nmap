  ## Installation

  *Steps required to install project and how to get the development environment running:*

  git clone https://github.com/AlexBlackeyed/GUI-Nmap.git
  cd GUI-Nmap
  sudo su
  pip3 install -r requirements.txt OR pip3 install ttkthemes
  python3 main.py
